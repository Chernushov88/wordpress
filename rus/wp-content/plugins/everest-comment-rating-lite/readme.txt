<p><strong> Everest Comment Rating Lite </strong> is a Free Wordpress Plugin that provides like and dislike features along with the rating counter for default WordPress comment. The buttons and counter can be enabled or disabled as per the requirement of the user.</p>

<p>This plugin simply expands the connection with the WordPress comments by empowering likes and dislikes buttons and easily shows comment rating of your users. It is also a fully responsive and easy-to-use plugin with an intuitive interface. Note that one needs to be logged in first in-order to like or dislike the comment via this plugin.</p>

== Installation ==

1. Unzip everest-comment-rating-lite.zip
2. Upload all the files to the /wp-content/plugins/everest-comment-rating-lite
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on Comments option menu in WordPress left admin menu and go to Everest Comment Rating Lite sub menu and configure all settings as per your requirement.

== Frequently Asked Questions ==

= What does this plugin do? =
This plugin allows user to add like and dislike features along with the rating counter on default WordPress comment. The buttons and counter can be also enabled or disabled as per the requirement of the user.

= Are Customization options available? =
Yes, You can customize your icons and counter as per your requirement. However, four different templates for like/dislike icon and counter each are also provided.
